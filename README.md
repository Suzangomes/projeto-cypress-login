#Projeto de Testes Automatizados com Cypress

#Sobre o Projeto
Este projeto contém testes automatizados utilizando Cypress.  
O objetivo é validar o comportamento de uma funcionalidade de login, garantindo que o sistema responda corretamente a cenários inválidos.


#Tecnologias Utilizadas
- Cypress  
- JavaScript  
- Node.js  


#Cenários Testados
- Preenchimento de campo de e-mail  
- Submissão de formulário  
- Validação de dados inseridos  
- Teste com entrada inválida
