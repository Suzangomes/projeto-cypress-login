# Projeto de Testes Automatizados - Login com Cypress

## 📌 Sobre o Projeto

Este projeto contém testes automatizados de login utilizando o Cypress.
O objetivo é validar o comportamento da funcionalidade de autenticação, garantindo que o sistema responda corretamente para cenários válidos e inválidos.

---

## 🛠 Tecnologias Utilizadas

- Node.js
- Cypress
- JavaScript
